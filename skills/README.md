# KC Skills

Custom skills for Klaus Claude framework.

Skills are invoked via `/kc:skill-name` pattern.

## Available Skills

*No skills defined yet*

## Adding New Skills

1. Create `[skill-name].md` in this directory
2. Follow Claude Code skill format (YAML frontmatter + markdown body)
3. Test with `/kc:skill-name`
4. Document in this README

## Skill Naming Convention

- Use `/kc:*` prefix to distinguish from SuperClaude skills (`/sc:*`)
- Examples: `/kc:parallel`, `/kc:optimize`
